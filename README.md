# UWTutor

### Required Instillations
* Docker / Docker-compose
* mongodb-community
* python 3.9 / pip
* NodeJS >= 14.15.3

### Server Setup
* Build: `docker-compose build`
* Run: `docker-compose up server`
* Test: `docker-compose run server-tests`

### Client Setup
* `cd client && npm i && npm start`
